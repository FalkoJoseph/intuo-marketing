# Wagon

Locomotive wagon

### Requirements

- [rbenv](https://github.com/rbenv/rbenv)
- [wagon](https://github.com/locomotivecms/wagon)

### Usage

    $ bundle install
    $ bundle exec wagon serve

Visit `http://localhost:3333`
